# usaps/homebrew-tap

Homebrew tap for [CMD-I](https://github.com/usaps/CMDI) — high-speed media offloading for film sets.

## Install

```bash
brew tap usaps/cmdi
brew install cmdi
```

This installs two commands: `cmdi` and `command` (alias).

## Uninstall

```bash
brew uninstall cmdi
brew untap usaps/cmdi
```

---

*This tap distributes the compiled macOS universal binary. Source code remains private.*
